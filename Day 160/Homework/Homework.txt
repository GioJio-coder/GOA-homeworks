replaceOne უკეთესია, რადგან ოპერაცია ერთ ნაბიჯში სრულდება,
delete და insert არის ორი ცალკე ნაბიჯი და შეიძლება მათ შორის შეცდომა მოხდეს
delete, insert შემთხვევაში id იცვლება. ერთი ოპერაცია ყოველთვის ჯობია ორს
ნაკლები რისკია მონაცემების დაკარგვის, თუ delete შესრულდა და insert ვერ შესრულდა, მონაცემი დაიკარგება.



db.products.deleteOne({ id: 1 })

db.products.deleteOne({ price: { $gt: 50 } })

db.products.deleteOne({ in_stock: false })

db.products.deleteOne({ rating: null })

db.products.deleteOne({ tags: "sale" })


db.products.deleteMany({ price: { $lt: 100 } })

db.products.deleteMany({ in_stock: false })

db.products.deleteMany({ rating:  false  })

db.products.deleteMany({
  price: { $gt: 150 },
  in_stock: false
})
